# Predictive Maintenance AI Dashboard

A beginner-friendly machine-learning web application that predicts whether a machine is likely to fail based on operating conditions.

The app uses the AI4I 2020 Predictive Maintenance Dataset and trains a Random Forest classifier with five input features:

- Air Temperature
- Process Temperature
- Rotational Speed
- Torque
- Tool Wear

## Project Structure

```text
predictive-maintenance-app/
├── app.py
├── train_model.py
├── requirements.txt
├── README.md
├── data/
│   └── ai4i2020.csv
└── model/
    └── predictive_maintenance_model.pkl
```

The `data/` and `model/` folders are created automatically when you run the training script.

## How It Works

1. `train_model.py` downloads the AI4I 2020 dataset from the UCI Machine Learning Repository.
2. The script selects five beginner-friendly machine parameters.
3. A Random Forest model is trained to predict machine failure.
4. The trained model, feature list, feature importance values, and evaluation metrics are saved to a `.pkl` file.
5. `app.py` loads the saved model and provides a Streamlit dashboard for real-time predictions.

## Setup Instructions

Install the required Python packages:

```bash
pip install -r requirements.txt
```

Train the model:

```bash
python train_model.py
```

Run the Streamlit app:

```bash
streamlit run app.py
```

## User Inputs

| Input | Description |
| --- | --- |
| Air Temperature | Temperature of the surrounding air near the machine, measured in Kelvin. |
| Process Temperature | Operating temperature generated by the manufacturing process, measured in Kelvin. |
| Rotational Speed | Speed of the rotating component, measured in RPM. |
| Torque | Rotational force applied by the machine, measured in Nm. |
| Tool Wear | Total tool usage time, measured in minutes. |

## Prediction Outputs

The dashboard displays:

- Machine Failure Risk: Yes or No
- Prediction Confidence
- Risk Level: Low, Medium, or High
- Failure probability score
- Feature importance chart
- Model accuracy and ROC AUC

## Notes for Beginners

The model file is not included until you run:

```bash
python train_model.py
```

If the Streamlit app says the model is missing, stop the app, train the model, and run Streamlit again.
